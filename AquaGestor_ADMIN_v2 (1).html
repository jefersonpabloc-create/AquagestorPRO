ℹ️ Como Funciona

Integração Firebase Firestore

  • Os usuários são criados via Firebase Authentication e seus metadados (nome, perfil, status) salvos na coleção users/{uid} do Firestore.
  • Apenas contas com campo perfil: "admin" no Firestore podem acessar este painel.
  • Ao criar um usuário aqui, ele é registrado no Firebase Auth e no Firestore simultaneamente.
  • Inativar um usuário define ativo: false no Firestore — ele não consegue mais usar o app (o app principal valida esse campo).
  • A senha do admin padrão é definida por você no Firebase Console → Authentication.
  • O log de atividade é salvo na subcoleção adminLogs do Firestore.
