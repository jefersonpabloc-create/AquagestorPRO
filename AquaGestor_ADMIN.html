ℹ️ Instruções de Uso

Como funciona o sistema de login:

  • Os usuários são armazenados no localStorage do navegador sob a chave __aquagestor_users__.
  • Para que o login do AquaGestor PRO funcione, ambos os arquivos (.html) devem ser abertos no mesmo navegador e no mesmo domínio (ex: ambos via file:// localmente, ou hospedados juntos).
  • O usuário padrão criado automaticamente é admin com senha admin123.
  • Recomendamos criar um novo administrador e alterar a senha padrão após o primeiro acesso.
  • Usuários inativados não conseguem fazer login no app principal.
  • A sessão do usuário expira automaticamente após 12 horas.
