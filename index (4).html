🔌 Conectar ao Supabase
Insira os dados do seu projeto Supabase. Eles ficam salvos no dispositivo — você configura uma única vez.
Supabase Dashboard → Settings → API → Project URL

🗄️ Criar Tabelas Automático
Clique no botão para criar todas as tabelas necessárias no seu banco de dados Supabase automaticamente.
